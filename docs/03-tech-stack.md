# The Tools We Use (Tech Stack)

If we were building a house, these would be our hammers, nails, and blueprints. We chose the newest and best tools available today to make sure Cognis is fast, beautiful, and never breaks down.

## The Visual Website (Frontend)
This is everything you can see, touch, and click.
- **Next.js & React**: The main blueprint we use to build the screens. It’s the same tool used by massive companies like Netflix and TikTok.
- **Tailwind CSS**: Our paintbrush. It lets us easily add gorgeous colors, sleek borders, and smooth animations.
- **Vercel**: The piece of land on the internet where our website lives. It makes sure the site loads instantly no matter where you are in the world.

## The Brain Behind the Scenes (Backend)
This is the invisible engine that processes data, makes decisions, and runs calculations.
- **Python**: The language our engine speaks. It’s famous for being great at Artificial Intelligence and heavy data crunching.
- **FastAPI**: The specific engine model. Just like its name says, it is incredibly fast at handling thousands of tasks at once.
- **Railway**: The powerhouse where our invisible engine runs 24/7.

## Storage (The Memory)
- **Supabase**: Our secure cloud vault. It stores all the user passwords, profiles, and messages safely so they are never lost.
