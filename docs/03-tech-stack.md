# Technology Stack

Cognis relies on a modern, bleeding-edge technology stack to maximize performance, maintainability, and visual fidelity.

## Frontend
- **Framework:** Next.js (App Router paradigm) with React
- **Language:** TypeScript
- **Styling:** Tailwind CSS (Vanilla utilities with extensive custom theme tokenization)
- **Icons:** Lucide React
- **Deployment:** Vercel

## Backend
- **Framework:** FastAPI (Python)
- **Language:** Python 3.12
- **Data Serialization:** Pydantic
- **Deployment:** Railway
- **Package Management:** `pip` / `uv`

## Infrastructure & Persistence
- **Database:** Supabase (PostgreSQL)
- **Storage:** Supabase Storage
- **Local Persistence:** HTML5 `localStorage` (for instant-rendering cache layers)

## Third-Party Integrations
- Crunchbase APIs (Sourcing)
- GitHub APIs (Talent Identification)
- HackerNews / Devpost Scrapers (Live Sourcing Signals)
