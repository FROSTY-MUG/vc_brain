# Cognis

Cognis is the intelligence engine for equitable capital. It acts as an ultra-fast, dynamic platform bridging founders and investors.

## Overview
Cognis aims to reduce the friction in discovering, analyzing, and funding early-stage companies. It leverages ultra-low latency UIs, parallel agent pipelines, and dynamic real-time data streaming to synthesize internet footprints into actionable sourcing signals.

## Getting Started
1. Clone the repository.
2. Ensure you have Node.js and Python 3.12 installed.
3. Install frontend dependencies: `npm install`
4. Install backend dependencies: `cd backend && pip install -r requirements.txt`
5. Run the frontend: `npm run dev`
6. Run the backend: `cd backend && uvicorn main:app --reload`

## Structure
- `/src`: Frontend Next.js application
- `/backend`: Python FastAPI application
- `/docs`: Comprehensive project documentation
