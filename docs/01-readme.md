# Welcome to Cognis!

Think of Cognis as a super-smart matchmaker. It helps brilliant startup founders meet the right investors, and helps investors find the next big thing before anyone else does.

## What Does It Do?
Usually, finding startups or investors takes months of emailing, browsing LinkedIn, and attending events. Cognis changes that. It constantly scans the internet (like Twitter, news sites, and coding platforms) to find signs of great companies being built. Then, it instantly shows these opportunities to investors on a beautiful, lightning-fast screen.

## How to Start It Up
If you want to run this app on your own computer, just follow these simple steps:
1. Make sure you have the basics installed (Node.js for the website, and Python for the brain behind the scenes).
2. Open your terminal (command prompt).
3. Start the visual website by typing: `npm install` and then `npm run dev`
4. Start the brain (backend) by typing: `cd backend` then `pip install -r requirements.txt` and finally `uvicorn main:app --reload`

## Where is Everything?
- The **`/src`** folder holds all the visual stuff—buttons, colors, and the screens you click on.
- The **`/backend`** folder holds the brain—the math and rules that decide who matches with whom.
- The **`/docs`** folder holds these simple explanation guides!
